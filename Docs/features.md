# 功能手册

## 快速开始

```bash
npm run tauri dev
```

构建生产版本：

```bash
npm run tauri build
```

### 命令行参数

支持在启动时直接传入路径，自动加载图片：

```bash
# 单张图片
imggroup.exe "C:\photo.png"

# 多张图片
imggroup.exe "C:\a.jpg" "C:\b.png"

# 文件夹（递归扫描子目录）
imggroup.exe "C:\Photos"

# 混合（图片 + 文件夹）
imggroup.exe "C:\photo.jpg" "C:\Photos"
```

参数之间用空格分隔，`--` 开头的参数会被过滤。

### 多语言切换

- **位置**：顶部工具栏 → 语言切换按钮（中文 / EN）
- **行为**：点击后立即切换界面语言，选择持久化到 localStorage
- **支持语言**：简体中文、English
- **自动检测**：首次启动根据系统语言自动选择（`navigator.language` 以 `zh` 开头则用中文）

## 功能列表

### 1. 加载文件夹

- **位置**：顶部工具栏 → 「📂 加载文件夹」
- **行为**：打开原生文件夹选择对话框，**递归扫描**所选目录及其所有子目录中的图片
- **分组**：加载的图片按文件夹层级构建树状分组，根节点为文件夹名
- **加载反馈**：点击后**立即显示**骨架占位格（pulse 动画），扫描完成后再替换为真实图片

### 2. 选择图片（多文件）

- **位置**：顶部工具栏 → 「🖼️ 选择图片」
- **行为**：打开多选文件对话框（已过滤图片格式），可一次选择多个图片文件
- **分组**：加载后归入「📄 临时分组」

### 3. 追加 / 重置模式

- **位置**：顶部工具栏 → 「🔄 重置」 / 「➕ 追加」
- **重置模式**（默认）：每次加载替换所有已有分组和图片
- **追加模式**：保留已有分组，新加载的文件夹作为新分组追加到列表末尾

### 4. 分组显示

- **位置**：底部控制栏 → 「📁 分组显示」开关
- **开启**：按文件夹层级以可折叠树状面板展示
- **关闭**：所有图片平铺为扁平网格视图

### 5. 树状折叠

- **分组根节点**：点击分组标题栏 **▶/▼** 切换折叠整个分组
- **子文件夹节点**：点击行 **▶/▼** 展开或折叠该文件夹
- **一键折叠/展开**：底部控制栏的「🔽 展开全部 / 🔼 折叠全部」按钮，**递归**影响所有子节点

### 6. 删除

- **删除分组**：点击分组标题栏末尾的 **✕** 按钮，移除整个分组及其图片
- **删除子节点**：鼠标悬停在文件夹节点行时显示 **✕** 按钮，点击后移除该文件夹及其所有子文件夹下的图片

### 7. 网格独立比例

每个网格（每个文件夹的图片集合）的格子宽高比以该网格**第一张图片**的原始宽高比为准，同一网格内所有格子比例一致。

## 大图查看器

点击任意网格图片打开全屏大图查看器。

### 8. 图片浏览

| 操作               | 效果                              |
| ------------------ | --------------------------------- |
| 点击网格图片        | 打开大图查看器                    |
| ◀ ▶ 侧边按钮       | 上一张 / 下一张（循环）           |
| 键盘 ← →           | 上一张 / 下一张                   |
| ESC                | 关闭查看器                        |
| 滚轮               | 缩放 (±0.1, 0.1x~10x)            |
| 拖拽图片           | 可拖到其他程序（如微信、PS）      |

**导航范围**：
- 开启分组显示时 → 在当前文件夹节点内循环
- 关闭分组显示时 → 全局循环

### 9. 三种显示模式

| 模式     | 按钮     | 效果                            |
| -------- | -------- | ------------------------------- |
| 适应窗口 | 适应窗口 | `object-fit: contain` 完整显示   |
| 短边适应 | 短边适应 | `object-fit: cover` 铺满裁剪    |
| 原始大小 | 100%     | 1:1 像素，可滚轮缩放            |

**缩放百分比**：显示实际缩放比例（含基础缩放 + 滚轮叠加）

### 10. 操作栏（hover 到底部显示）

| 按钮                    | 功能                              |
| ----------------------- | --------------------------------- |
| 100% / 适应窗口 / 短边适应 | 切换显示模式                     |
| 隐藏信息 / 显示信息     | 切换悬浮信息栏                    |
| 📋 元信息               | 打开 IPTC + EXIF 元数据弹窗       |
| ⛶ 全屏                 | 切换窗口全屏                      |
| 📂 在资源管理器中显示   | 在文件管理器中定位该文件          |
| 🖥️ 默认方式打开         | 用系统默认程序打开                |
| ✕ 关闭                 | 关闭大图查看器                    |

### 11. 悬浮信息栏

显示在图片底部，毛玻璃效果，不干扰操作。

格式：`[序号/总数] 文件名 — 宽×高 — 文件大小 — 缩放%`

示例：`[3/12] DSC_001.jpg — 1920×1080 — 2.3 MB — 67%`

### 12. 元信息弹窗

点击「📋 元信息」打开，展示该图片的：

- **IPTC 字段**：Caption、Keywords、ByLine、CopyrightNotice 等
- **EXIF 字段**：Make、Model、DateTimeOriginal、FNumber、ISO、UserComment 等

支持「📋 复制全部」一键复制所有字段文本。

### 13. 全屏模式

点击「⛶ 全屏」切换窗口全屏/窗口模式，使用 Tauri 原生窗口 API。

## 快捷键汇总

| 按键      | 上下文       | 效果             |
| --------- | ------------ | ---------------- |
| ←         | 大图查看器   | 上一张           |
| →         | 大图查看器   | 下一张           |
| ESC       | 大图查看器   | 关闭查看器       |
| 滚轮 ↑/↓  | 大图查看器   | 放大/缩小        |

## 交互细节

| 交互                     | 效果                     |
| ------------------------ | ------------------------ |
| 点击分组标题栏           | 折叠/展开该分组          |
| 点击文件夹节点行         | 折叠/展开该文件夹        |
| 悬停网格项               | 显示图片文件名           |
| 悬停文件夹节点行         | 显示 ✕ 删除按钮          |
| hover 到底部区域         | 显示操作栏               |
| 点击背景（大图查看器）   | 不关闭（防止误触）       |
| 点击 ✕ 或 ESC            | 关闭大图查看器           |

## 状态提示

| 场景                     | 显示内容                     |
| ------------------------ | ---------------------------- |
| 首次打开                 | 欢迎页 + 操作引导            |
| 加载中                   | 骨架占位格 pulse 动画        |
| 文件夹无图片             | 错误横幅提示                 |
| 用户取消对话框           | 静默处理，无提示             |
| 图片无元数据             | 弹窗显示"不含元数据信息"     |

## 支持的图片格式

`jpg` / `jpeg` / `png` / `gif` / `bmp` / `webp` / `svg` / `ico`
